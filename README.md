# MyFund

IDEA插件,用于随时在Idea中查看基金最新估值.摸鱼神器.

部分功能参照了 [leeks](https://github.com/chris-tao/leeks) ,但是leeks没有实现估值估算亏盈,且不支持最新的IDEA,因此重新开发.

**持有份额表示购买的基本份额数,而不是购买的总金额**
**周末计算收益选中框不选中,表示在周末的时候不会出现盈亏(周末休市)**
